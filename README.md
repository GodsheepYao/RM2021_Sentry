## 二、功能列表

1. 延迟启动
2. 电机掉线看门狗
3. 电机上电或杀死自动恢复
4. 摩擦轮过热保护（未实现）
5. 卡弹保护
6. 向PC上报电机信息
7. 与上云台上板进行通信
8. 上下云台遥控器切换
9. 光纤放大器轨道定位
10. 编码器测量轨道长度
11. 底盘撞柱检测
